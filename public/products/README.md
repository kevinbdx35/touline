# Images des Produits

Placez vos photos de toulines dans ce dossier avec les noms suivants :

## Noms de fichiers requis :

1. **touline-classique.jpg** - Photo de la Touline Classique
2. **touline-premium.jpg** - Photo de la Touline Premium
3. **touline-personnalisee.jpg** - Photo de la Touline Personnalisée
4. **mini-touline.jpg** - Photo de la Mini Touline
5. **touline-decoration.jpg** - Photo de la Touline Décoration
6. **set-toulines.jpg** - Photo du Set Toulines

## Recommandations :

- **Format** : JPG ou PNG
- **Dimensions recommandées** : 800x800 pixels minimum
- **Orientation** : Carré de préférence (ratio 1:1)
- **Qualité** : Bonne résolution, bien éclairée
- **Fond** : Neutre de préférence (blanc ou clair)

## Notes :

- Les images s'affichent avec un effet de zoom au survol
- Si une image est manquante, un placeholder s'affichera
- Vous pouvez aussi utiliser le format .png si vous préférez
